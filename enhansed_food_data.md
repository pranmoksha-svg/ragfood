# 🍽️ Enhanced Food Database Showcase

The food database was significantly enhanced to improve the performance of the RAG system. A total of **20 diverse food items** were added, covering multiple cuisines, health-focused options, and comfort foods.

---

## 🌍 Dataset Diversity

The dataset includes:

- **8 World Cuisine dishes**
  - Example: Pad Krapow Gai (Thai), Falafel (Middle Eastern), Bibimbap (Korean), Moussaka (Greek)
  
- **6 Healthy food options**
  - Example: Chia Pudding, Grilled Chicken Salad, Smoothie Bowl
  
- **6 Comfort food dishes**
  - Example: Mac and Cheese, Chicken Curry, Ramen, Lasagna

This diversity allows the system to handle a wide range of user queries.

---

## 🧾 Rich Data Structure

Each food item contains detailed structured information:

- Name and category  
- Description (75+ words)  
- Ingredients  
- Cooking method  
- Nutritional benefits  
- Cultural background  
- Dietary tags  
- Allergen information  

This structured format improves retrieval accuracy and context quality.
